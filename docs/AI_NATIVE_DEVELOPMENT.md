# AI-native development disclosure

AgentSeal is an AI-native open-source project.

The project was built through human-directed, multi-model AI orchestration: multiple AI systems were used during design exploration, implementation iteration, documentation drafting, testing, bug discovery, and release-hardening workflows.

The maintainer remains responsible for the project direction, code review, release decisions, documentation, and security posture. AI assistance does not replace review, testing, reproducibility checks, or evidence standards.

## What this means

- AgentSeal was developed with AI-assisted engineering workflows from the start.
- Multiple AI systems were used as collaborators for reasoning, code review, test generation, debugging, and documentation.
- The project should be judged by its source code, methodology, tests, reports, and reproducible behavior.
- Public reports generated by AgentSeal should be reviewed critically before being used as evidence.

## What this does not mean

- It does not mean users should trust the tool without inspecting reports.
- It does not mean every generated report is automatically correct.
- It does not mean AI systems are maintainers of the project.
- It does not remove the need for human review before publishing audit claims.

## Contribution policy

Contributors may use AI-assisted development tools, but submitted changes should be reviewed, tested, and explainable. Security-sensitive code, token handling, evidence-link rendering, dataset downloading, and report methodology changes should receive extra scrutiny.
