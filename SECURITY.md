# Security policy

## Reporting vulnerabilities

Please open a private security advisory or contact the maintainers directly before public disclosure.

## Secret handling

Do not include GitHub/HuggingFace tokens in issues, logs, screenshots, or reports.
AgentSeal should redact token-like values in the TUI, but users should still treat exposed tokens as compromised and revoke them.

## Evidence URLs

Broken, synthetic, or placeholder URLs must not be rendered as clickable evidence in reports.
